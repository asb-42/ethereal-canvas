# Model Cache Directory

This directory contains downloaded models for Ethereal Canvas.

## Structure
- `Qwen-Image-2512/` - Text-to-Image generation model
- `Qwen-Image-Edit-2511/` - Image editing model

## Usage
Models are automatically downloaded to these directories during installation or on first use.

## Storage
These models are large (several GB each) and should be backed up if possible to avoid re-downloading.
